# STICKERS — Adesivos do zine

Coloque imagens aqui com os nomes abaixo e eles aparecerão
automaticamente nos 4 slots dos cantos da Home:

- `home-1.png` → canto superior ESQUERDO
- `home-2.png` → canto superior DIREITO
- `home-3.png` → canto inferior ESQUERDO
- `home-4.png` → canto inferior DIREITO

Extensões aceitas (em ordem de preferência): `.png`, `.webp`, `.jpg`, `.jpeg`

## Dicas para bons adesivos

- **Formato:** quadrado ou próximo disso (o slot tem 92×92 px no mobile).
- **Fundo:** PNG com fundo transparente funciona melhor, mas o slot
  tem fundo de papel off-white, então qualquer imagem serve.
- **Conteúdo:** fotos de manobras suas, logos de marcas que curte,
  colagens, recortes de zine. Nada muito detalhado — no tamanho
  pequeno, imagens simples se destacam mais.
- **Não precisa girar a imagem:** os slots já aplicam rotações
  diferentes por posição pra dar a sensação de "colei torto".

## Como testar

1. Adicione `home-1.png` nesta pasta.
2. Recarregue o app no navegador (Ctrl+F5 no PC, ou fecha e abre
   de novo no celular).
3. O adesivo aparecerá no canto superior esquerdo com rotação
   leve e fita crepe amarela virtual por cima.

Se não aparecer nada, verifique:
- O nome está exato? (case-sensitive em alguns servidores)
- A extensão é uma das aceitas?
- O arquivo não está corrompido?

Enquanto os slots estão vazios, aparece o placeholder discreto
com o kanji 瞬 (shun) num círculo pontilhado.
